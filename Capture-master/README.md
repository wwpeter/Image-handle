# Capture

> 图像视频处理，现在线上的产品:[爱拍美图] (https://appsto.re/cn/ti6t7.i), 自定义相机，用了GPUImage实时滤镜，还有一些图片的裁剪，拼图，装饰处理，现在主要是用swift写的。

## 安装：
 
```
git clone https://github.com/dulingkang/Capture.git
git submodule update --init --recursive
```
